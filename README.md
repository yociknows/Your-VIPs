# ✨ 遇貴人｜您的尊爵人脈金庫 (Your-VIPs)

這不僅是一份資料庫，更是您社交圈的智慧導航儀。

### 🚀 核心功能
- **💰 人脈存款系統**：將每一份情誼量化為資產，看見您的人脈實力。
- **🔥 社交氣溫監測**：自動提醒聯繫，讓關係永不結冰。
- **🔮 流年開運指南**：結合五行與節氣，提供精準的送禮與問候建議。
- **📅 慶生行事曆**：貴人壽辰不遺漏，讓溫情傳遞最及時。

### 📱 如何安裝至手機 (PWA)
1. 使用手機瀏覽器開啟此頁面。
2. **iPhone (Safari)**：點擊「分享」圖示 -> 選擇「加入主畫面」。
3. **Android (Chrome)**：點擊選單 -> 選擇「安裝應用程式」。
